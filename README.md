# HWP 지원서 자동작성

관공서마다 다른 HWP 지원서/이력서 양식에 같은 개인정보, 자격증, 경력, 학력 정보를 반복 입력하지 않도록 도와주는 로컬 앱입니다.

기존 이력서 HWP에서 내 정보를 추출하고, 기관에서 받은 HWP 양식을 분석한 뒤, 사용자가 중간값을 수정하면 최종 HWP 파일을 자동으로 생성합니다.

## 프로젝트 개요

이 프로젝트는 구직자가 여러 기관의 HWP 지원서 양식을 작성할 때 반복되는 입력 작업을 줄이기 위한 Windows 로컬 도구입니다. 문서와 개인정보를 외부 서버로 보내지 않고 사용자의 PC 안에서 처리하는 것을 기본 방향으로 합니다.

핵심 흐름은 다음과 같습니다.

1. 기존 HWP 이력서에서 재사용 가능한 프로필 정보를 추출합니다.
2. 새 지원서 또는 기관별 HWP 양식을 분석합니다.
3. 항목별 추천 입력값을 보여주고 사용자가 수정합니다.
4. 확인된 값을 원본 HWP 양식의 표와 입력 칸에 맞춰 채웁니다.
5. 완성된 HWP 결과물을 로컬 파일로 저장합니다.

## 주요 기능

- HWP 이력서에서 이름, 연락처, 이메일, 주소, 학력, 경력, 자격증 추출
- 기관별 HWP 양식 분석
- 중간값 추천 및 직접 수정
- HWP 표 칸에 맞춘 자동 입력
- 오늘 날짜, 성명, 개인정보 동의 체크 자동 입력
- 각 PC에 설치해서 로컬로 실행
- Python/pywin32를 내장 워커 exe로 포함

## 사용 흐름

1. 앱을 실행합니다.
2. 기존 이력서 HWP 파일을 업로드해 프로필 정보를 추출합니다.
3. 작성할 지원서 HWP 양식을 업로드해 입력 칸을 분석합니다.
4. 앱이 추천한 입력값을 검토하고 필요한 부분을 수정합니다.
5. `HWP 생성`을 눌러 완성된 지원서 파일을 내려받습니다.

## 사용자 PC 필수 조건

- Windows
- Windows용 한글(HWP)

Python 3와 pywin32는 사용자가 따로 설치하지 않아도 됩니다.

## 개발 실행

```powershell
npm install
npm start
```

브라우저에서 `http://127.0.0.1:5177`을 엽니다.

## 데스크톱 앱 미리보기

```powershell
npm run desktop
```

## HWP 워커 exe만 만들기

```powershell
npm run build:worker
```

## Windows 설치파일 만들기

```powershell
npm run build:win
```

생성된 설치파일은 `dist` 폴더에 저장됩니다.

```text
dist\HWP 지원서 자동작성 Setup 0.1.3.exe
```

자세한 배포 방법은 `INSTALL.md`를 참고하세요.
