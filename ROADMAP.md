# Roadmap

HWP 지원서 자동작성 프로젝트의 개선 계획입니다.

## 현재 단계

- HWP 이력서에서 기본 프로필 정보 추출
- 기관별 HWP 지원서 양식 분석
- 추천 입력값 검토 및 수정 화면 제공
- 로컬 서버 기반 웹 UI와 Electron 데스크톱 실행 지원
- Windows용 한글(HWP) 자동화를 위한 Python 워커 구성

## 다음 목표

- 다양한 HWP 양식에 대한 매핑 정확도 개선
- 추출된 프로필 정보의 항목별 신뢰도 표시
- 지원서 작성 전 누락 항목과 확인 필요 항목 안내
- 생성 결과 파일의 오류 검증 절차 강화
- 설치 파일 배포 흐름 정리 및 사용자 안내 문서 개선

## 중장기 방향

- 자주 쓰는 프로필 항목 저장과 불러오기
- 기관별 양식 템플릿 재사용
- 작성 이력 관리
- 더 친절한 오류 메시지와 진단 화면
- HWP 자동화 실패 시 복구 절차 개선

## 제외하거나 신중히 다룰 범위

- 개인정보를 외부 서버로 전송하는 기능
- 사용자의 동의 없는 문서 자동 업로드
- HWP 프로그램이 설치되지 않은 환경에서의 완전한 문서 생성

## 우선순위

1. HWP 양식 분석 안정성
2. 생성 결과물 검증
3. 사용자 입력 검토 화면 개선
4. 설치 및 배포 편의성
5. 템플릿 재사용 기능
