# HWP 지원서 자동작성 설치/배포 안내

이 앱은 각자 자기 Windows PC에 설치해서 쓰는 로컬 앱입니다. 서버에 문서를 올리지 않고, 업로드한 HWP와 생성된 HWP 결과물은 각 PC 안에 저장됩니다.

## 사용자 PC에서 필요한 것

사용자 PC에는 아래 1가지만 준비되어 있으면 됩니다.

1. Windows용 한글(HWP)

Python 3와 pywin32는 설치파일 안에 포함된 `hwp_worker.exe`가 대신 처리하므로 사용자가 따로 설치하지 않아도 됩니다.

## 왜 한글(HWP)은 필요한가요?

이 앱은 기관에서 받은 HWP 양식의 표, 칸, 줄간격, 서식은 그대로 둔 채 빈칸만 채웁니다. 이 작업은 한글 프로그램의 HWP 자동화 엔진을 사용해야 안정적으로 처리할 수 있습니다.

그래서 Python은 없애도 되지만, 한글(HWP) 프로그램 자체는 사용자 PC에 설치되어 있어야 합니다.

## 개발/빌드 PC에서 설치파일 만들기

```powershell
npm install
npm run build:win
```

`npm run build:win`은 아래 작업을 자동으로 수행합니다.

1. `scripts/hwp_worker.py`를 `bin/hwp_worker.exe`로 패키징
2. Electron 앱 안에 `hwp_worker.exe` 포함
3. Windows 설치파일 생성

완료되면 `dist` 폴더에 설치파일이 생성됩니다.

```text
dist\HWP 지원서 자동작성 Setup 0.1.3.exe
```

## 사용자에게 전달할 파일

사용자에게는 아래 설치파일만 전달하면 됩니다.

```text
dist\HWP 지원서 자동작성 Setup 0.1.0.exe
```

사용자는 설치파일을 실행한 뒤 바탕화면 또는 시작 메뉴에서 `HWP 지원서 자동작성`을 실행하면 됩니다.

## 사용자 PC 환경 점검

문제가 생겼을 때만 아래 스크립트로 확인하면 됩니다.

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\check_prerequisites.ps1
```

점검 내용:

- 내장 HWP 워커 exe 포함 여부
- Windows용 한글(HWP) COM 등록 여부

## 사용자 PC에서 쓰는 방법

1. 설치파일을 실행합니다.
2. `HWP 지원서 자동작성`을 실행합니다.
3. 기존 이력서 HWP를 넣어 내 정보를 추출합니다.
4. 기관에서 받은 지원서/공고 HWP를 넣습니다.
5. 중간값을 확인하고 필요한 항목을 수정합니다.
6. `HWP 생성`을 누르면 칸에 맞춰 입력된 HWP 결과물이 나옵니다.

## 자주 막히는 경우

- 앱은 켜지지만 HWP 생성이 실패함: 사용자 PC에 Windows용 한글(HWP)이 설치되어 있는지 확인하세요.
- 한글 보안 알림이 뜸: 한글 자동화가 실행되는 정상 과정일 수 있습니다.
- 회사 보안 PC에서 설치가 차단됨: 내부 보안 정책에 따라 설치파일 허용 또는 코드 서명이 필요할 수 있습니다.
- 여러 사람이 같이 쓰고 싶음: 설치파일을 각 PC에 설치하면 됩니다. 문서와 개인정보는 각자 PC에만 저장됩니다.
